# Task Manager App

A full-stack Task Manager web application built using React (Vite) and Node.js (Express).

---

## Features

- Add tasks
- Delete tasks
- Task metadata:
  - Title
  - Description (Notes)
  - Priority
  - Due Date
- Task categorization:
  - Today’s Plan
  - Important Tasks
  - Quick Tasks
- Theme switching (5 themes)
- Responsive UI
- Full-stack API integration

---

## Tech Stack

### Frontend
- React (Vite)
- CSS (custom styling)

### Backend
- Node.js
- Express.js
- In-memory storage
